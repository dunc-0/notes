1. Protocols

  • Simple Service Discovery Protocol - SSDP

    • a network protocol based on the Internet protocol suite for advertisement and discovery of network services and presence information. The example below is from my phone searching for castable devices.

    Wiki dial-multiscreen-org
  • Multicast DNS - MDNS

    • multicast DNS (mDNS) protocol resolves hostnames to IP addresses within small networks that do not include a local name server.

    wiki
  • DNS - Refresher

    • DNS points to root name servers, bounces to TLD name servers, queries ANS NSO returns an ip address

  • Dynamic Host Configuration Protocol - DHCP

    • Static - Manually assigned ip address can also include subnet, default gateway, and dns Dynamic - Automatically assigns ip addresses, subnet mask, default gateway and dns. Brodcasts a request so will appear on wireshark
